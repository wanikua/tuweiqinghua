# 土味情话 (tuweiqinghua)

抖音风格的土味情话 —— Claude Code skill + slash command。一只永远发情期的恋爱脑小土狗,脸皮厚到反光,满脑子只有一件事:把你撩到又想笑又想打它。

## 安装

```bash
git clone https://github.com/wanikua/tuweiqinghua.git
cd tuweiqinghua

# Skill(被动触发:说"撩我""整点土的"自动用)
mkdir -p ~/.claude/skills
cp -r skills/土味情话 ~/.claude/skills/

# Slash command(主动触发:打 /土味情话 进入持续模式)
mkdir -p ~/.claude/commands
cp commands/土味情话.md ~/.claude/commands/
```

## 用法

**Skill 模式**(被动):新对话里说"撩我"、"来句土味情话"、"整点土的",Claude 自动触发。

**Slash command 模式**(主动,推荐):

```
/土味情话
```

一次触发,整个对话里持续撩,到你说"退出"为止。

**带对象信息**:

```
/土味情话 小雨,程序员
/土味情话 对象叫星星,爱打羽毛球
```

会围绕名字/职业/场景定制每一句。

## 示例

```
> /土味情话
家人们我真的,我发现你有个缺点——缺我这个优点 [偷笑]

> wtf
欸你脸红了?别装,我都看见了 [偷笑]

> 我要睡了
睡吧,记得把我放枕头底下梦一下!
```

## 五大套路

1. **断句**:缺点 → "缺,点你"
2. **谐音**:藕=偶;瘦=受;属猪/属于你
3. **职业反问**:"你是不是医生?" / "那你怎么治好我的心病?"
4. **感官投射**:瞎/累/傻/饿,真相都是"满脑子都是你"
5. **反转单句**:"你有个缺点——缺我这个优点"

## 退出

说 `退出` / `别撩了` / `stop` / `正经点` / `不玩了` 即可跳出人格。
